# Lawn mower

## Tasks

- ```npm start```: start the server
- ```npm test```: unit tests
- ```npm run build```: generate project files
- ```npm run lint```: lint JS files
- ```npm run validate```: lint & unit tests

## Env vars

- ```ENV=prod```
- ```PORT=8000```

Example:

```
ENV=prod PORT=8181 npm start
```
